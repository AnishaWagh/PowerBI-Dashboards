📊 Project Overview

This project is an interactive Power BI Dashboard that analyzes the "PhonePe Pulse" dataset. It provides a deep dive into the evolution of digital payments in India, tracking how millions of users transition from traditional cash to UPI and digital wallets across different states and districts.
🔍 Key Objectives

    Transaction Trends: Visualize the growth of transaction volumes and values over time.

    User Demographics: Map the geographical distribution of PhonePe users across India.

    Device Analysis: Identify which mobile brands are most commonly used for digital payments.

    State-wise Performance: Compare digital adoption between different Indian states and districts.

📁 Repository Structure
🛠️ Tools & Technologies

    Power BI Desktop: For data modeling and creating the interactive dashboard.

    Power Query: For cleaning and transforming the raw JSON/CSV data.

    DAX (Data Analysis Expressions): For calculating custom metrics like "Average Transaction Value" and "Growth %".

💡 Potential Insights

    Trend: Which quarter usually sees the highest spike in transactions (Festive season effect)?

    Top State: Which state leads in transaction volume versus transaction value?

    Category: Are "Peer-to-Peer" transfers growing faster than "Merchant Payments"?
